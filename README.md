# kubernetes-troubleshooting-lab

This repository contains common Kubernetes troubleshooting scenarios, investigation steps, root cause analysis, and resolutions.

## Troubleshooting Scenarios

| Scenario | Status |
|-----------|----------|
| Node NotReady | ✅ |
| ImagePullBackOff | 🚧 |
| CrashLoopBackOff | 🚧 |
| Certificate Expiry | 🚧 |
| Ingress Issues | 🚧 |

## Objective

The goal of this repository is to document practical Kubernetes troubleshooting scenarios and provide structured investigation and resolution procedures.
